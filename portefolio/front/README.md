# Ulysse Mercadal Portfolio

A simplified typography portfolio featuring interactive animations on the name "Ulysse Mercadal" using Nanum Myeongjo font.

## Features
- **Wireframe (Bezier Bubble)** animation
- **Repetition 3D (Size Waterfall)** animation
- **Cross 3D (Shadow Skew)** animation
- Light/Dark mode toggle

## Tech Stack
- Next.js
- Paper.js
- Opentype.js
- D3.js (for scaling)
